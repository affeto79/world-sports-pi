Politique de confidentialité

Dernière mise à jour : 2026

World Sports Pi est une application développée pour l’écosystème Pi Network. Le respect de votre vie privée est une priorité.

Données collectées

World Sports Pi utilise uniquement les données fournies par Pi Network lors de l’authentification, notamment :

  • Nom d’utilisateur Pi

Aucune donnée sensible (localisation, contacts, fichiers personnels) n’est collectée ni stockée par World Sports Pi.

Utilisation des données

Les données sont utilisées exclusivement pour :

  • L’authentification de l’utilisateur
  • L’accès aux fonctionnalités de l’application

Partage des données

World Sports Pi ne vend, ne loue et ne partage aucune donnée personnelle avec des tiers.

Sécurité

Nous appliquons des mesures raisonnables pour protéger les données fournies via Pi Network.

Contact

Pour toute question, veuillez contacter le développeur via Pi Network.

En utilisant World Sports Pi, vous acceptez cette politique de confidentialité.
